# Primsa SQLite Todo App

### Simple todo **API**

## Features

- Create todo
- Lists all todos
- List only one Todo
- Update todos
- Delete todos

Basically a **CRUD** backend application

## Technologies used

- NodeJS
- ExpressJS
- Prisma
- SQLite 3

## What was learnt

- Middleware
- Request/Response
- Data modeling with prisma (So wonderful)

Built this application just to shaper my backend skills
